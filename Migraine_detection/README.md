# Migraine Detection Application

## Overview
The Migraine Detection Application is a web-based tool designed to assist users in predicting migraine types based on user input or predefined profiles. The application utilizes a machine learning model to provide predictions and relevant information about different migraine types.

```
└── README.md
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd migraine-detection-app
   ```

2. **Install Dependencies**
   Ensure you have Python and the required libraries installed. You can install the necessary libraries using pip:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Application**
   Start the application using the following command:
   ```bash
   python app.py
   ```
   The application will be accessible at `http://127.0.0.1:5000`.

## Usage Guidelines
- Upon accessing the application, users can choose to enter values manually or select from predefined migraine profiles.
- The application will display the predicted migraine type along with relevant information based on the input provided.
- Users can interact with the interface to explore different migraine types and their characteristics.
